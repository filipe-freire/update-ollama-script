# Linux Ollama Update Script

A custom bash script that updates your Ollama version on linux. Until Ollama supports automatic updates, this script offers a convenient way to update your Ollama version.

## Prerequisites

- Ollama installed on your system
- You might need to mark the script as executable by running the following command:

```bash
chmod +x ./update-ollama
```

**If you're updating from a previous version:**

1. Check that Ollama has been updated by running the following command:

```bash
ollama -v
```

## Running the script

```bash
sh ./update-ollama
```

## Official Docs

For the official docs on how to install/update ollama, please refer to <https://ollama.com/download/linux>
